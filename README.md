# Meadowlark Travel

Cайт, который связывает путешественников с местными экскурсоводами-любителями и сотрудничает с фирмами, выдающими напрокат велосипеды и мотороллеры и предлагающими туры по окрестностям с акцентом на экотуризм.


## Технологический стек

Backend-проект на Node.js + Express с базовой структурой, REST API, обработкой ошибок и файлом .env.


## Локальный запуск

Клонирование проекта

```bash
git clone https://github.com/hazar84/meadowlark.git
```

Переход в директорию проекта

```bash
cd meadowlark
```

Установка зависимойстей

```bash
npm install
```

Запуск проекта

```bash
npm run start
```


## Запуск тестов

Чтобы запустить тесты, выполните следующую команду:

```bash
npm run test
```
